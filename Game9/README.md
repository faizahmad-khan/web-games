# Pacman Game

A classic Pacman game implemented in HTML, CSS, and JavaScript. Navigate through the maze, collect dots while avoiding ghosts!

## Features

- Classic Pacman gameplay with maze navigation
- Dot collection mechanics
- Ghost enemies with basic AI movement
- Score tracking system
- Lives system
- Level progression
- Responsive controls for smooth gameplay

## Controls

- **Arrow Keys** → Move Pacman in four directions
- **Space** → Pause/resume game

## Game Elements

- **Pacman**: The player character that navigates the maze
- **Dots**: Small points to collect for scoring
- **Power Pellets**: Special dots that allow Pacman to eat ghosts
- **Ghosts**: Enemies that chase Pacman
- **Score**: Points earned by collecting dots and eating ghosts
- **Lives**: Number of attempts before game over

## Implementation

The game uses HTML5 Canvas for rendering and implements core Pacman mechanics including:
- Maze navigation with collision detection
- Ghost AI with different movement patterns
- Power pellet mechanics
- Score calculation
- Level progression
- Lives system

## Files

- `index.html`: Main game page structure
- `style.css`: Visual styling and layout
- `script.js`: Game logic and mechanics

## How to Play

1. Click the "Start Game" button to begin
2. Use arrow keys to navigate Pacman through the maze
3. Collect all dots to advance to the next level
4. Avoid ghosts or use power pellets to eat them
5. The game ends when all lives are lost
