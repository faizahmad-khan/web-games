# Rock Paper Scissors Game

A classic Rock Paper Scissors game built with HTML, CSS, and JavaScript. Play against the computer in this fun and interactive game!

## How to Play

1. Choose your move: Rock, Paper, or Scissors
2. The computer will randomly select its move
3. The winner is determined by the classic rules:
   - Rock beats Scissors
   - Paper beats Rock
   - Scissors beats Paper
4. The first player to reach 5 points wins the game

## Features

- Clean and responsive design
- Score tracking
- Win/loss detection
- Modern UI with gradient backgrounds
- Interactive elements with hover effects

## Technologies Used

- HTML5
- CSS3 (with gradients and animations)
- JavaScript (ES6)
## How to Run

1. Clone or download this repository
2. Navigate to the Game6 directory
3. Open `index.html` in your preferred web browser

Alternatively, you can play the game directly by opening the [game page](index.html).


Alternatively, you can run a local server:

```bash
cd Game6
python -m http.server 8000
```

Then visit `http://localhost:8000` in your browser.
